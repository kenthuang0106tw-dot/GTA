# GTA City Prototype

一個可直接在瀏覽器執行的俯視角城市動作遊戲原型。

## 開啟方式

直接用瀏覽器打開 `index.html`。

## 功能

- 近戰攻擊
- 射擊
- 可上車/下車與駕駛
- 路人 NPC 會走動並對危險反應
- 大型城市地圖、道路、建築、車輛與小地圖
- 支援鍵盤滑鼠與手機觸控按鈕

## 電腦操作

- `WASD` / 方向鍵：移動或開車
- 滑鼠：瞄準
- 滑鼠左鍵：射擊
- `F`：近戰攻擊
- `E`：上車 / 下車
- `Space`：車輛加速
- `R`：補滿彈藥

## 手機操作

- 左下虛擬搖桿：移動 / 開車
- `拳`：近戰攻擊
- `砰`：射擊
- `車`：上車 / 下車
- `油`：車輛加速
